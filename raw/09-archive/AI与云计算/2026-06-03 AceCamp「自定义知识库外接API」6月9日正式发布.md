# AceCamp「自定义知识库/外接API」6月9日正式发布

**日期**: 2026-06-03 | **来源**: acecamptech.com | **公司**:  | **标签**: TMT

## 核心观点
- 如果您有自己私有部署的 Agent/OpenClaw，我们开放对接 API；
- 如果没有，来本营知识库: 一站式解决外部数据（公开+半公开）的聚集、筛选、跟踪与展示。

## 正文

*线上使用说明会火热报名中: *[*https://www.acecamptech.com/eventDetail/60535892*](https://www.acecamptech.com/eventDetail/60535892)

*活动时间: 2026年6月10日（周三） 20:00-20:30*

我们将于 2026年6月9日 正式推出全新产品: **自定义知识库** + **外接 API 服务**。

在本营知识库中，您可以:

**1、轻松监控自媒体，以及授权我们汇集您拥有的第三方数据:**

![](https://image.acecamptech.com/article_image/20501063/0.2684346084429564.png)

*（如您有特别关注的数据源想纳入监控范围，请通过销售或扫描本文末尾的二维码与我们联系。）*

**2、在“内容广场”一站式浏览来自不同平台的最新资讯:**

![](https://image.acecamptech.com/article_image/20501063/0.6776260731677898.png)

**3、关注公司、行业，AI 自动筛选相关信息存入您的个人专属知识库:**

![](https://image.acecamptech.com/article_image/20501063/0.40469177226893094.png)

![](https://image.acecamptech.com/article_image/20501063/0.8653130788208858.png)

****4、在首页添加“自定义卡片”，告诉 AI 您最想看到的内容是什么**，AI 为您自动生成结果、可选加入 AceCamp 日报推送:**

![](https://image.acecamptech.com/article_image/20501063/0.6008652319043086.png)

**5、我们提供 **外接 API**，支持您本地的 Agent 直接查询本营研究和知识库内容，省去 Agent 部署中繁琐的数据接入、整合工作。**

![](https://image.acecamptech.com/article_image/20501063/0.9749352826184846.png)

AI 为大家带来了数据消费习惯的改变，但无论是 Chatbot、Agent 还是龙虾，未来的工具应用都需要立足于完整的底层数据聚合。我们推出 自定义知识库 + 外接 API 将助力大家高效完成数据汇聚、筛选、追踪，让我们的客户可以专注于最终决策。

您可以通过销售或扫描以下二维码与我们沟通:

- API 使用申请

- 定制数据源需求接洽

- 更多 AI 能力需求交流
