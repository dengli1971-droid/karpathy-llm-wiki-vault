## 核心摘要

智谱 GLM 5.1 发布后到 2026 年底预计还有 2 次小版本迭代，核心方向是 Coding + Agent。当前 Coding 领先 Kimi/DeepSeek/千问，但千问 3.6 + DeepSeek 4 追赶迅速（千问把 API 推广作为 2026 战略，动员阿里云全销售）。**算力卡瓶颈实质化**：新增英伟达卡需等高层访问活动结束（隐含中美博弈窗口），原有云租赁（腾讯/阿里/华为/火山）只能保持存量，新增无法满足。计算卡 30-40% 训练 + 60-70% 推理。国产适配方向：昇腾 + 沐曦。**模型变现压力**：付费用户实际选择 > 排行榜测评，市场竞争更看口碑。

## 关键数据点

- **后续迭代**：2026 年底前 2 次小版本
- **训练/推理拆分**：30-40% / 60-70%
- **国产适配**：昇腾 + 沐曦
- **竞争格局**：智谱 Coding 仍领先；千问 API 战略
- **算力增量瓶颈**：英伟达卡新批等高层访问后
