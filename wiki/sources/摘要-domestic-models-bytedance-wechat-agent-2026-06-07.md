## 核心摘要

11 个非共识结论：**#1 字节 2026 采购类 Capex ¥4,300-4,500 亿**（vs 市场 ¥2,000-3,000 亿预期，**+50%**），超过其他头部厂商总和；GPU 整机 ¥1,700-1,800 → 2,300 亿 / CPU 服务器 750 → 1,100 亿 + 全年服务器 90 → 125 万台 + IDC + 组网。**GPU 拆解**：海外 1,200 亿（NVIDIA 65% + AMD 30% + 博通/高通自研 5%）+ **国内 1,100 亿**（NVIDIA 210 亿 + 寒武纪 250 亿 + 昇腾 ~230 亿）；**国产 GPU 占字节 2026 预算 10% → 2027 15% → 2030 目标 30-35%**（十年渐进非爆发）。**#2 卖方市场到位**：字节通算服务器交付 4 周 → 8-10 周；百度 4 周 → 20+ 周；**A100/A800 月租仍 ¥20,000+/张**（4-5 年旧卡）打破折旧预期；行业恐慌前置采购。**#3 字节日 token 指数加速**：2025/1 5T → 11 月 50T → 2026/6 200T → 8 月 500T → **12 月 1,000T**；月付费客户 5 → 30 万家；2025/2 训练/推理需求交叉 → 当前推理主导。**#4 微信 Agent 本质交易闭环不是聊天**：用户意图 → 微信支付商家信用评分推荐小程序 → 全链路自动唤起下单支付；三大变现：**支付抽佣 + 小程序广告 + 技能分成**（不接熟人社交）；首批 2,000 高频小程序（美团/大麦/猫眼/小红书）；目标 MAU 破亿；商业逻辑接近淘宝/美团模式而非豆包/元宝。**#5 腾讯 AI 投入广义 2025 ¥600 亿**（含人才/数据/训练/推理/营销）**→ 2026 ¥720-780 亿**（vs 市场窄口径 ¥360-400 亿）；2025 整体 Capex ¥800 → 900+ 亿。**#6 混元 3 预览不发正式 → 直接跳混元 4**（2026Q4，进度顺利可能提前；1T 参数对标 ChatGPT 4o）；四大提升：编程能力 + MoE 动态激活 + 原生 Agent 工具调用（无需 MCP）+ 多模态图文融合；刘煜宏全流程主导未离职。**WorkBuddy**：MAU 80-90 万（上线两月）/ 职场 45% / **付费率 3-4%**；多模型 MiniMax/GLM/DPSK 前三（开源部署腾讯算力无需付费）；安全沙箱（首创）+ Skill Hub + CodeBuddy 积分体系 + 企业/微信打通；腾讯 Agent 四层矩阵 WorkBuddy + DataBuddy + 马维斯 + 混元 Memory。**#7 MaaS 综合毛利仅盈亏平衡**（vs 市场预期 20-30% SaaS 类比）：文本 5-6 个点 / 视频 12 / 语音图像 6-8 / 算力租赁 20-25%（5 年长约）；**未分摊训练成本仅推理**；盈亏平衡价：文本 ¥60/亿 token / 音频 ¥120 / 视频 ¥200；GPU 折旧是核心成本（电力占比低）；**GPU 涨价 10% → MaaS 毛利接近零**。**#8 字节定价策略不是降价是让竞品亏损**：定在自身有毛利但竞品同价亏的价位；DPSK infra 强 → "同定价字节亏但 Kimi 可盈利"；智谱 coding 涨价证明差异化可脱离价格战。**#9 MiniMax 技术路径风险被低估**：全而不精；基座始终未超过智谱；非大厂容错低 3-6 月难纠偏；当前估值最高 AI 独角兽之一面临调整风险。**#10 国产芯片性能未突破**：绝大多数高价 + 勉强可用；选择是替代而非领先；CPU 无出口禁令替代慢于 GPU；MS 性价比视角 vs 产业实际感受温差，投资者需双视角。**#11 中小模型不会被价格战淘汰**：技术终局未到（世界模型/VLA/多模态/蛋白质/分子模拟分叉），细分方向有优势可生存 2-3 年；MAU/ARR 只影响估值不影响生存。**总投资含义**：算力卖方市场云厂商利润弹性低估 + 持有存量 GPU 的阿里云/腾讯云/火山引擎隐藏利润 + 算力租赁 token 转售涨价直接受益 + 字节体系训练已解决/瓶颈是推理 → 推理优化能力强厂商（Kimi infra）受益 + 腾讯估值需重新定价（微信 Agent 应类美团/阿里本地生活逻辑而非 ChatBot）。
