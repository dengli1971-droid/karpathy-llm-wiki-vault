## 核心摘要

服务器电源三级架构：(1) **一次电源**：220V/380V AC → 48V DC（机柜外）；(2) **二次电源（板载电源砖）**：48V → 12V；(3) **三次电源 POL**：12V → 1V GPU/CPU。**HVDC 演进**：±400V/800V DC 直接进入机柜，前端 DCDC 替代 ACDC，**2026H2 小批量试用**。**ADI 二次电源套片方案**：主控 + 功率 MOS + DRIVER 三件套，新雷能采用 + 自购磁芯/PCB 组装电源砖。**英伟达供应商**：GB300/Rubin 平台方案无区别，用量随 GPU 功耗。主流单砖 2,000W，1,500W + 3,000W 共存，3,000W 即将量产。**磁芯供应商**：铂科新材（深圳）供新雷能 + 伟创力。**HVDC 主供**：台达 + 光宝 + 维谛 + 麦格米特 + 欧陆通。整体看 2026H2 国产二次电源迎窗口期。

## 关键数据点

- **三级架构**：一次 48V / 二次 12V / 三次 ~1V
- **HVDC 时间**：2026H2 小批量
- **电源砖功率**：1,500W / 2,000W / 3,000W
- **ADI 套片**：主控 + 功率 MOS + DRIVER
- **磁芯供应**：铂科新材
- **HVDC 主供**：台达 / 光宝 / 维谛 / 麦格米特 / 欧陆通
