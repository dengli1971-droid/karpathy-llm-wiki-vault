## 核心摘要

腾讯 WorkBuddy 内部视角。**定位**："桌面数字员工"To B 生产力，C 端仅引流（中国 C 端 AI 订阅不可跑通：付费意愿低/客单低/留存差）；核心价值=每天替企业员工省 2 小时可重复劳动 + 可量化 ROI。**商业化节奏**：2026/3 上线公测、4 月起收入（企业版 ¥78/人）→ 5/15 涨价 **¥198/人**（同时 Token 单价提升）；5-12 月企业锁单+行业方案落地。**渠道**：腾讯云存量直客 + 集成商/代理商最高 30% 返佣；下半年深耕政企/制造/互联网高毛利行业。**技术壁垒**：①Token Hub 多模型（混元 + DeepSeek V4 + 智谱 GLM + Kimi + MiniMax）切换无需自带 API Key；②深度集成企业微信组织架构 → 数据隔离/Token 配额/合规审计/私有化（信息安全 2.0）；③小程序云/本地双模式 + Cloud Boot 微信原生集成；④架构沿用 CodeBuddy 1+ 年稳定性。**vs 竞品**：日活 = 字节飞书系 2-3×；vs 阿里通义万相 Copilot 也领先；目标 2026 成为中国最大企业 AI 办公订阅、长期 To B 业务核心现金流。**资源**：腾讯 Agent 矩阵 = WorkBuddy + CodeBuddy + DataBuddy + 马维斯 + 混元 Memory；WorkBuddy 战略优先级最高、算力随取。**与微信打通**：当前小程序+Cloud Base 双向通信；2026 下半年规划打通微信内消息/支付/聊天/朋友圈/公众号 → 统一"微信 Agent"。**企业微信底盘**：MAU 3 亿、客户 100 万+；渗透低（100 客户企业可能只开 20 账号）。

## 关键数据点
- 企业版价：¥78 → 5/15 涨至 ¥198/人
- 代理返佣：最高 30%
- Token Hub 内置：混元/DPSK V4/GLM/Kimi/MiniMax 等
- 日活 = 字节飞书系 2-3×
- 企业微信 MAU 3 亿 / 客户 100 万+
- 渗透：100 客户企业仅 20 账号
- 行业聚焦：政企 / 制造 / 互联网
