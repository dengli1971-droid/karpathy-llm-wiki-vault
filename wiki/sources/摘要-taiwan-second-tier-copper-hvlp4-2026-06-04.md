## 核心摘要

**HVLP4 供需缺口持续 2026-2029**：2026 六家主供产能仅 1,000 吨 vs 月均万吨潜在需求 → 终端被迫降规；2027 技改后 3,000 吨；至 2029 缺口持续。**六家通过认证**：日本三井+福田+古河 / 台湾金居 / 欧洲卢森堡 / 中国铜冠 → 第七家德福（2026/5 刚过台光电+松下认证）+ 第八家台湾长春（2027 有望）→ **第九家看不到**。**HVLP4 良率普遍低**核心因表面粗糙度极低与电镀传动轴摩擦力不够 → **后处理线设备瓶颈**：日本三船（龙头）+ New Long + 西工业全部排产至 2028 实际 2029-2030 已订满；过去一年设备价格 **涨近 3 倍**；国内泰金/宏天/钛金仅能做 HVLP1-2。**德福 2026/5/27 宣布扩 5 万吨电子电路铜箔全部 HVLP**（特别 3-4 代），向三船订 **24 条后处理线 ≈ 三船 3 年总产能 + 交付周期 4-6 年**。**HVLP4 加工费**：龙头 ¥240/kg；下半年三井可能从 ¥240 → ¥500/kg；但日企细水长流不会大幅暴力涨价（参考超薄载体铜箔 3 月涨 <15%）；预期温和上调。**国内梯队加工费差距**：铜冠 ¥15 万/吨 vs 三井 ¥24 万；德福不到头部一半（量产后向 ¥25 万靠拢）。**HVLP1/2 加工费**（人民币 vs 三井等头部）：¥90-95/kg / ¥11 万/吨 / 16 万/吨 / 24 万/吨。**HVLP1-2 转移给国内**：三井 2027 大概率停产 HVLP1，金居明年停 HVLP2，卢森堡通知客户停一代；承接者德福、宝鼎（山东金宝）、诺德。**HVLP4 2027 总供应**：3,000 吨（三井 1,000-1,200 + 金居 500 + 铜冠翻倍 + 福古卢翻倍 + 德福新增 + 长春 200+）；前三家三井+金居+铜冠占 ~2,000 吨；其他 1,000 吨。**载体铜箔结构性机会**：三井 20 年 95%+ 份额；2026 起 AI HBM 载板 + 1.6T+ MSAP → 产能不够 → 下游被迫评估二供 → 德福+方邦 2026H2 量产；金居明确**不进**（工艺/设备/客户认证三重障碍）。**HVLP5 难推**：HVLP4 接近铜箔电性天花板；CCL 转向树脂创新（PPO/PTFE/碳氢 M10）+ HVLP4 铜箔 + Low DK Two 玻璃布组合突破。**M8/M9 头部 CCL**：台光电（最大）+ 台耀 + 生益 + 斗山 + 松下；联茂/南亚落后；台光电近年扶持金居+铜冠两家。**HVLP 客户 97-98% 为 CCL 厂**。

## 关键数据点

- **HVLP4 供需**：2026 1,000 吨 / 2027 3,000 吨 vs 月需万吨
- **HVLP4 加工费**：¥240/kg → 可能 ¥500/kg（温和上调 +15%）
- **国内 vs 头部加工费**：铜冠 ¥15 万 vs 三井 ¥24 万；德福 <12 万
- **HVLP1-3 加工费**：¥90-95/kg / 11 / 16 万/吨
- **日本三船后处理线**：2026-29 产能全订满，价格 +3×
- **德福 2026/5/27 扩产**：5 万吨全 HVLP，24 条三船线 = 3 年产能
- **三井历史**：HVLP4 提价频率低（载体铜箔 3 月 <15%）
- **2027 HVLP4 主供**：三井 1,000-1,200 / 金居 500 / 铜冠翻倍 / 福古卢翻倍 / 德福新增 / 长春 200+
- **载体铜箔三井份额**：95%+
- **M8/M9 CCL 头部**：台光电/台耀/生益/斗山/松下
