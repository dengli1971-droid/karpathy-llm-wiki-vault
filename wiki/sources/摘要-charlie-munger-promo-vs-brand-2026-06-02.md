## 核心摘要

芒格系列第二篇，提出**判断「促销」是营销武器还是价格战死亡螺旋**的 3 个问题框架。谷物公司常年促销但仍赚钱，因促销是品牌系统里的战术工具；航空公司常年促销则毁灭价值，因消费者只认价格。差异不在「促销与否」，而在**「优惠券撕掉之后消费者还认不认这个牌子」**。这是 [[芒格思维模型]] 的具体应用案例，也跟 [[Beta_Alpha锚点框架]] 形成方法论级互补：品牌偏好 = 业务侧 Alpha 锚点；只剩促销 = 周期性 Beta 颠簸。

## 三个判断问题（核心框架）

1. **促销之后，消费者会不会永久迁移？**
   - 活动结束回原品牌 → 促销 = 营销成本
   - 一旦发现低价替代品就永久离开 → 促销 = 价格战起点

2. **企业有没有能力在促销之外恢复价格？**
   - 强品牌：周期性促销 + 合适时机涨回
   - 弱品牌：一旦降价就再涨不回来 — 这是品牌质量最重要的测试

3. **促销是否改善了长期关系？**
   - 提升复购 / 扩大会员 / 货架曝光 / 教育新品类 → 短期牺牲利润换长期关系
   - 清库存 / 抢份额 / 维持收入指标 → 训练消费者只认低价

## 关键判别

> 真正值得警惕的不是「促销很多」，而是「除了促销没有别的武器」。
> 如果一家企业不能通过品牌 / 渠道 / 产品 / 服务 / 评价 / 习惯 / 网络效应留住客户，每一次促销都可能变成下一轮价格战的起点。

## 在 AI / 半导体行业的映射应用

- **AI 应用层「以补贴换 token 调用」**：MaaS 厂商如果不能在订阅模式 / 企业级合约 / 工作流锁定 / 数据网络效应留客，token 降价就是价格战的起点
- **国产 GPU 集群 vs 英伟达**：如果国产卡只是「便宜+月租优惠」而没有 CUDA 替代生态 / 客户工作流绑定 / 数据网络效应，迟早进入价格战死亡螺旋
- **大模型订阅竞争**：DeepSeek 低价、Kimi 免费冲量、智谱降价——优惠期结束后用户是认品牌还是认便宜，决定了哪些是真好生意
