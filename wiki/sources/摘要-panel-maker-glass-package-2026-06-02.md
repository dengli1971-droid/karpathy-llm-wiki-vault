## 核心摘要

京东方+康宁四方向战略合作（与已 ingest [[摘要-microled-optical-bydone-corning-2026-06-01]] 互补）：每季度技术/供应链/战略全层级交流。康宁上月与 Nokia 签光纤大单后希望延伸到先进半导体封装 + 光通信。**京东方中试线 2026H2 投片**；静态样品已送国内客户评测，未到量产验证。**三种产品形态**：① 过程载板（提供半成品 + RDL 重布线层 = 参与部分封装环节）② PCB 类基板（替代 ABF 载板）③ 中介层。**康宁原片非货架货**：组分按京东方工艺调整，解金属/玻璃热失配。康宁有意在大陆体系将光纤业务 + 京东方玻璃载板/Micro LED 有源光模块整合共同卖给国内数据中心客户（**CPO 层面更深合作空间**）。**全球玻璃基板量产路径**：① **三星 2018 年首次量产**（智能手表 AP 用玻璃中介层）② 谷歌第四代 TPU 曾有玻璃中介层产品但因工艺能力问题未延续 ③ Amkor + SKC 等中期验证 ④ 英特尔玻璃基板用于 CPU 的 EMIB 硅桥玻璃版本（类似 CoWoS 但器件不同）。**台湾对标**：友达 + 富采控股 + 微软仍 demo 阶段；力成科技 MCU/射频已用但未到 GPU；群创卖了一个厂自身无突破；台积电 InFO_oS 推到 2028 年才工程化验证 CoWoS 之上。**京东方先发优势**：10+ 年前已探索玻璃基封装 + 改造北京五代线 + 扇出型面板级封装经验丰富；中介层与同业同步起步。**光通信**：京东方尚未确定合作对象，国内 CPO 等测试刚起步，基于 LED 非显示衬底 + 外延片 + 玻璃载板/Micro LED → 光交换环节整合。

## 关键数据点

- **中试线 2026H2 投片**
- **三种产品形态**：过程载板（含 RDL 半成品）/ ABF 替代基板 / 中介层
- **三星玻璃中介层量产里程碑**：2018 年（智能手表 AP）
- **谷歌第四代 TPU**：玻璃中介层试过未延续
- **英特尔**：玻璃 EMIB 硅桥替代
- **台积电 InFO_oS**：2028 年工程化验证
