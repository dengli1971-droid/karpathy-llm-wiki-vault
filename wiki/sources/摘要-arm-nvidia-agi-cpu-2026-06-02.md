## 核心摘要

Arm CEO Rene Haas + 黄仁勋联合 Computex 2026 演讲。**Arm AGI CPU 上线**（台积电生产）：两种系统级方案——风冷 36 kW × 8,000 核 / 液冷 200 kW × 45,000+ 核；每机架性能 2x x86（同功耗 2x 性能或同性能 -50% 功耗），预计为行业节省 10 GW 电力 + 100 亿美元。**Agentic AI 驱动 CPU 倍增**：未来同功耗 CPU 核数 +4-8 倍（甚至更高），CPU 5 年 TAM 1,000-1,200 亿美元 → 可能翻倍。**关键产业事件**：Google TPU 8t/8i 头节点 CPU 从 x86 转向自研 Axion（Neoverse 架构）—— 同等性能 -60% 功耗；Amazon Graviton 新设计项目超半数选 Graviton（vs 几年前 0%）；NVIDIA Vera 广泛合作伙伴支持。生态系统 100% 台湾构建（TPU 机架/NVIDIA 机架/Graviton）。新增 AGI CPU 客户 Oracle + ByteDance。Arm 提供三层方案：IP 授权 / CSS 子系统 / AGI CPU 成品。

## 关键数据点

- **AGI CPU 系统**：风冷 36kW × 8,000 核；液冷 200kW × 45,000+ 核
- **vs x86 优势**：同功耗 2x 性能 / 同性能 -50% 功耗
- **TPU Axion**：同性能 -60% 功耗
- **CPU TAM 5 年**：1,000-1,200 亿美元，可能翻倍
- **CPU 核数需求**：同功耗 +4-8x
- **AGI CPU 客户**：Meta / Rebellions / SAP / Cerebras / OpenAI / SK Telecom / Oracle / ByteDance
- **台湾累计 Arm 芯片**：~2500 亿颗
