## 核心摘要

**非共识 Alpha**：CPO 不是 MLCC 终结者而是结构性升级催化剂。**总量净增 / 结构上移 / 价值集中**——单台 51.2T 交换机 MLCC 价值量从 $1,500-2,000（传统 64×800G）→ $3,000-8,000（CPO 版本），**翻倍乃至 4 倍**，前提是高端低 ESL 渗透按路线图兑现。**三层价值贡献**：① CPO 封装基板上反向几何 LICC / 3 端子 MLCC（紧贴 SerDes 接口与光引擎，ESL 比传统降 30-50%，单价 **3-5×** 传统 0402/X7R）② ASIC 封装背面 Landside 高容值小尺寸（三星电机路线图 0402≥47μF/0603≥100μF，目前 **路线图目标 + 样品指标**非量产）③ **硅电容**（Empower Semiconductor EC 系列十几 μF + ESL 个位数 pH = 比 MLCC 0.2-0.8 nH 低 2 个数量级 + 2.5 μF/mm² 容密度 + 嵌入基板核心工艺兼容；三星电机据报道获北美头部数亿至十亿美元订单）。**短期红利锁定村田 + 三星电机**：村田全球 MLCC 38-42% 份额 + 反向几何低 ESL 产线积累最深 + 与消费电子周期脱钩；三星电机硅电容布局激进 + 双时代订单（MLCC + 硅电容）。**A 股直接受益弱**（风华/国巨 vs 村田/三星电机存代差）但 **MLCC 介电陶瓷粉体 BaTiO₃**（量价逻辑纯粹）+ **CCL/高频板材**（低 Dk/Df）间接受益。**CPO 时间表**：2026 是验证期 / 2027 过渡期 / 2028 规模化兑现期；OIF CPO 标准未定稿；**封装良率 + 热管理 + 维修可靠性**是核心制约（液冷背板+精准温控代价高，光引擎与电芯片深度绑定单一组件失效整板报废）。**风险**：若退回 NPO 价值跃升打折；2028 后推迟 → 估值溢价压力；硅电容 2027 良率成本突破 → 直接竞争（现在比 MLCC 高 5-10×）。

## 关键数据点

- **传统 51.2T 交换机 MLCC**：800G 模块 150-250 颗 × 64 端口 × $6-10 = $400-650 模块侧；整台 $1,500-2,000
- **CPO 51.2T 交换机 MLCC**：$3,000-8,000（2-4× 增长）
- **800G 模块 MLCC**：150-250 颗 / $6-10
- **反向几何 LICC / 3 端子 ESL 降幅**：30-50%；单价 3-5× 传统 X7R
- **硅电容 vs 传统 MLCC**：ESL 2 个数量级低；容密度 2.5 μF/mm²；当前成本 5-10×
- **村田 MLCC 份额**：38-42% 全球
- **Broadcom Bailly 架构**：51.2T 量级；光互连功耗 -70%；能效 15 pJ/bit → ≤5 pJ/bit
- **CPO 路线图节奏**：2026 试验 / 2027 小批商用 / 2027-28 规模化（千台+/年）
