## 核心摘要

**Rubin Ultra（NVL576 die 计 / Kyber 机架 / H2 2027）正交背板 = 把 GPU 间高速信号从线缆改为 PCB 板内走线，消除线缆连接器阻抗不连续**。**单块正交背板 78 层 × 1.2m × 0.2m × 1cm 厚（量产无人区）**；单机柜 PCB 子系统价值 11.7 万美元（vs GB300 3.5 万 / Hopper 1 万），PCB 暴涨 233% vs GPU 涨 57%——剪刀差源于材料技术升级。**224Gbps PAM4 = 56GHz 奈奎斯特 + 铜导体趋肤深度 0.28μm**：HVLP4/HVLP5 镜面铜箔成为标配 + M9 Dk 3.0-3.5/Df 0.002-0.004 在 78 层超厚板局部热斑下偏紧 → **PTFE 成最强候选**（Dk 2.1-2.6 / Df 0.0002-0.001、宽频宽温稳定 + 高度对称分子结构 + C-F 键能 544 vs C-H 413 kJ/mol）。**三方案博弈**：M9+Q 布（极致性能但 Q 布钻孔 100-200 钻嘴 vs FR-4 1500-2000 + 日东纺垄断）/ M8+PTFE（性能差 3-4% 但成本 <1/3，加工最难良率最低）/ M10 PTFE（实验室）。**PTFE 加工三大噩梦**：(1) 冷流性与层压翘曲（320℃+ 极端物理应力、78 层纵向超微米对位偏差）/ (2) 孔壁润湿与金属化（PTFE 化学惰性 + 钠-萘络合物或等离子体活化危险昂贵 + 工艺窗口窄）/ (3) 钻孔拉丝胶渣 + HVLP4 镜面铜箔与 PTFE 树脂咬合力下降产生爆板。当前打样良率 ~70%，单块 78 层报废损失万美元级。**供应链双重卡脖子**：(1) **Q 布日东纺接近垄断**（高端 T-glass 单一口径）+ 2025/8 扩 3 倍但 2027-28 释放 + 2026 NVDA 单家需求 400-500 万米/月 vs 远低有效产能；(2) **M10 PTFE 树脂 = 大金/罗杰斯把控**（金属离子 ppb 级），国内圣泉 M9 碳氢树脂百吨级 / 东材+迪赛 BCB 千吨级试产；(3) **HVLP4 全球宽松**（400-500 吨/月 = 三井金属 + 国内金居各半），国内第五代铜箔进 M10 试样。**制造端**：欣兴 60-70%（compute tray）+ TTM 15%；switch tray 欣兴 40-55% + TTM 10-20%；胜宏/深南/沪士积极打样。**市场空间基准 290-367 亿 RMB（CoWoP/CPO 变量下可压缩到 145-193）**。**关键验证窗口 = 2026 Q3-Q4 NVDA code 批准 + 2026 Q4-2027 Q2 PTFE 良率从 50-60% 提升到 75%+ + 2027 H2 量产**。

## 关键数据点

- **正交背板规格**：78 层 / 1.2m × 0.2m × 1cm
- **单机柜 PCB 价值**：11.7 万美元（GB300 3.5 万 / Hopper 1 万 / +10x）
- **PCB 涨幅 vs GPU**：233% vs 57%
- **224Gbps 奈奎斯特**：56GHz / 趋肤深度 0.28μm
- **M9 vs PTFE 介电参数**：Dk 3.0-3.5/Df 0.002-0.004 vs Dk 2.1-2.6/Df 0.0002-0.001
- **M8+PTFE vs M9+Q 布 成本**：<1/3
- **Q 布钻嘴寿命**：100-200 孔（vs FR-4 1500-2000）
- **NVDA Q 布需求**：400-500 万米/月（2026）
- **HVLP4 全球月需求**：400-500 吨（三井金属/金居各半）
- **打样良率**：~70%（万美元级报废）
- **市场规模基准**：290-367 亿 RMB（CoWoP/CPO 下 145-193）
- **PTFE 加工温度**：>320℃
- **日东纺 T-glass 扩 3x 释放**：2027-28
