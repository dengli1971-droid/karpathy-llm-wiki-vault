## 核心摘要

国内 AI 芯片综合排序（单卡 + 互联）：**寒武纪 690 单卡最强**（算力 + 互联 + 综合集群），但 12-13 万定价偏高 → 性价比不突出；**昇腾 950 PR ~7 万元**；昆仑芯 P800 + 平头哥 PPU；2026H2 沐曦/深算四号有望对标 690。**互联能力 = 训练 + Agent 推理万卡集群关键**：(1) 昇腾因通信业务背景具备万卡级集群；(2) 寒武纪可支持数千卡（未到万卡）；(3) 二线厂商专注单卡互联不突出，PPU（平头哥）储备相对多。**未来 3 年 GPGPU vs ASIC**：国内市场 GPGPU 占主导——通用性更强 + 复用 NVIDIA 逻辑 + 生态成熟；ASIC 长远技术优势但生态运维要求高（Google 海外大投入构建生态；Meta/MSFT + 博通 ASIC 效果不理想）。寒武纪转 GPGPU 可能但艰难（推倒重来）。**CPU 应用价值量**：随相数 + 互联深度提升，每板 CPU 价值量持续上升。

## 关键数据点

- **寒武纪 690 定价**：12-13 万元
- **昇腾 950 PR**：~7 万元
- **互联能力**：昇腾 万卡 / 寒武纪 数千卡 / 平头哥 储备厚 / 二线弱
- **未来 3 年**：国内 GPGPU 主导
- **ASIC 失败案例**：Meta/微软 与博通定制效果不理想
