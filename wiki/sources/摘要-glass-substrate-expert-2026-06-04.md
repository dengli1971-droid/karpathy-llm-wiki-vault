## 核心摘要

**TGV 玻璃基板应用分两大领域**：① 显示（Micro LED 直显载板已批量，4-6 层板）② 半导体（Interposer / Glass Core / 射频 / 光模块）。**Micro LED vs 半导体 TGV 工艺差异两点**：通孔尺寸 + 填孔工艺。Micro LED 通孔 100μm + 玻璃 0.4-0.5mm + 深宽比 4-5:1（难度低）→ 用导电浆料（铜浆/铜包银浆/银浆）填孔（CTE 匹配 + 树脂粘接）+ 表面研磨 + PVD 镀铜；问题区域树脂上镀铜易剥离。**半导体 Interposer/Glass Core 需要**更精密结构。**国内 Micro LED TGV 玩家**：**沃格光电 + 三叠纪 + 兴森 PCB；京东方 2025 年底入局**双向显示 + 半导体。**Glass Core 应用进度**：CPO 进展更快（小批量），封装载板英特尔最快（高性能计算）。**国内 CPO 光模块玻璃基板**：所有项目小批量送样、测试结果良好但未到量产；除 TGV 还包含光波导技术（玻璃基板厂自做 vs 光模块厂自做两条路径）。**国内玻璃 Interposer 进展比海外更快**：目标替代 TSMC CoWoS 的 TSV 技术；客户中芯微电子 + H 客户 + 比亚迪；进度略早于 Glass Core 封装载板。**国内玻璃原片性能与海外差距大**——TGV 产业链大陆最大问题是材料对外依赖：
- 鹏辉玻璃：生产/测试量较大但稳定性 vs 肖特仍有差距
- 力诺 / 凯盛 / 山东药玻：1 年+ 送样调配方解决批次稳定性
- 戈碧迦：相对表现好
- 旗滨：新进入

半导体材料验证周期长，不同应用 CTE 等参数要求不同需针对配方 → 大厂倾向国际合作。**激光钻孔设备**：国际乐普科标杆；国内帝尔 + 大族 + 德龙 + 华工，国产基本可用满足 Interposer/Glass Core 需求，与国外差距远小于玻璃原片差距；帝尔/大族领先。**激光钻孔价值评估指标 = 效率**（每秒孔数，行业稳定 5,000/6,000/8,000）；不同产品所需设备数差异巨大。**4 层板产线投资 ¥6-10 亿/条 → 年产 10-15 万 m²**。**PVD 镀膜两种**：连续式（流水线高效大尺寸）+ 集群式（半导体周围分布腔体）；单价 ¥600+ 万 - ¥1,000+ 万。**裂纹检测 AOI**：玻璃透明传统 AOI 不适用 → 大多定制开发 + 进口依赖。**半导体玻璃原片粗片标准**：510×515mm，厚度 0.7mm 需减薄研磨抛光（许多厂不具备此能力）。

## 关键数据点

- **Micro LED TGV 通孔**：100μm + 玻璃 0.4-0.5mm + 深宽比 4-5:1
- **国内 Micro LED TGV 玩家**：沃格 / 三叠纪 / 兴森 / 京东方
- **国内玻璃 Interposer**：进度快于海外，客户中芯微+H+比亚迪
- **国内玻璃原片玩家**：鹏辉 / 力诺 / 凯盛 / 山东药玻 / 戈碧迦 / 旗滨（vs 肖特）
- **国内激光钻孔**：帝尔 / 大族 / 德龙 / 华工（vs 国际乐普科）
- **激光钻孔效率**：5,000-8,000 孔/秒
- **4 层产线投资**：¥6-10 亿/条 → 年产 10-15 万 m²
- **PVD 镀膜单价**：¥600 万 - ¥1,000+ 万
- **玻璃原片标准粗片**：510×515mm × 0.7mm
