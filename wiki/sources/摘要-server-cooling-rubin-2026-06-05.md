## 核心摘要

**GPU 散热演进 = 风冷→金属盖板 IHS→裸 Die + 相变 TIM→裸 Die + 石墨烯 TIM**：H 系列（700W）风冷可覆盖 4U 主板；GB200（>2000W）水冷必选 + 裸 Die + 霍尼韦尔有机相变材料 TIM；**Rubin 2026/5 初最终确定 = 石墨烯 TIM + 水冷板**（放弃液态金属铟镓合金 = 一致性/翘曲/不良率问题）。**冷板形态轮回**：GB200 单大冷板（GPU+CPU 三颗）→ GB300 多小冷板 → Rubin 回归大冷板。**冷板单价 ~150 美元**（GB200 高单价低良率 → GB300/Rubin 规模化降本平衡功率上行）。**微通道间距**：GB200/300 = 0.15mm → Rubin = 0.1mm（通道增 1/3、传热增 1/3，匹配功率提升）；**0.05mm 理想但行业未准备好**，3D 打印精度 0.15mm 也未能突破。**Rubin 水冷板供应商**：俊强 / AVC / 台达 / 健策 / 双鸿（铲齿工艺、技术不高门槛大量参与）；价值量 100-150 美元。**Rubin TIM 石墨烯供应商**：鸿富诚 / 富烯（常州）/ 海外 Parker；多家备选。**健策从台积电金属盖板合作积累 + 0.05mm 微通道打样虽未在 GB300 落地，仍进入 Rubin 备选寄望 Rubin Ultra/Feynman 工业化**。**前沿放弃路线**：台积电晶圆表面直接蚀刻微通道（热膨胀系数不匹配漏液风险）；**浸没式液冷 PUE 低但 PCB 信号丢失 + 流速不足 600-700W 上限，不再可行**。**GB200 IHS 单价 5 美元、霍尼韦尔 TIM 单颗 GPU ~3 美元**（仅低功耗芯片如 Adapter/Switch 保留 IHS）。

## 关键数据点

- **微通道间距**：GB200/300 0.15mm → Rubin 0.1mm（理想 0.05mm）
- **冷板单价**：~150 美元（Rubin 100-150）
- **GB200 IHS 单价**：5 美元
- **霍尼韦尔 TIM 单颗 GPU**：3 美元
- **H 系列 GPU 功耗**：~700W（风冷上限）
- **GB200 GPU 功耗**：>2000W
- **浸没液冷散热上限**：600-700W
