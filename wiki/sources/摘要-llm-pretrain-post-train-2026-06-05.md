## 核心摘要

**国内头部模型预训练 70-80% + 强化学习 ~10% + SFT 6-7%** 当前；**未来 SFT 降至 3-4% + 强化学习升至 15%**。预训练占比高的根本原因 = 国内模型参数规模仍小（~1 万亿）vs 海外 5-6 万亿；万亿模型预训练几十万亿 token 量 >> 后训练蒸馏的几百万-千万样本。**国内万亿+ 玩家 = DeepSeek V4 1.6T / 阿里 ~1.5T / 字节 1.7T**。**字节训练卡数十万张**：国内 H100 + 少量 B200 / 海外 H200 + B200。**模型架构创新方向**：Coding 能力 + 多模态统一模型 + MoE 负载均衡 + Attention 优化 + 长上下文。**中美差距正在缩小**：海外创新乏力（DeepSeek V2 影响力 << 早期版本），如果海外只是 scale-up 不会拉大差距；GPT-4 类 3-6T 参数 vs 国内 1.6-1.7T（vs 历史几千亿）。**国内核心瓶颈 = 算力 + Infra 而非数据**：万卡/5万/10万卡互联高效能 Infra 极难；数据可采买/合成/自有。算法不构成瓶颈（人才流动半年扩散）。**海外训练租赁可行**（20-30k 参数训练已转海外用 GB200，加速迭代效率），核心训练保留国内。**合成数据**：预训练当前 ~10% / 强化学习 50-60%，未来海外预训练 30-40% 是合理水平。**国内训练投入排序**：字节 > 阿里 > DeepSeek > 智谱/MiniMax（后两者不再推大模型）。

## 关键数据点

- **国内当前算力分配**：预训练 70-80% / RL ~10% / SFT 6-7%
- **未来分配**：SFT 3-4% / RL 15%
- **国内模型规模**：DeepSeek V4 1.6T / 阿里 1.5T / 字节 1.7T
- **海外模型规模**：5-6 万亿，下一步 10 万亿
- **GPT-4 级参数**：3-6 万亿
- **字节训练卡数**：数十万张（H100 + 少量 B200）
- **合成数据预训练占比**：当前 10% / 海外未来 30-40%
- **合成数据 RL 占比**：50-60%
- **国内 vs Mythos 能力差距**：~1 年+
