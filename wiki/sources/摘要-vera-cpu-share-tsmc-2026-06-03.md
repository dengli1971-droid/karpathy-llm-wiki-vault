## 核心摘要

NVIDIA 业绩会披露 Vera CPU 收入达 **200 亿美元**（订单性收入），覆盖 2026H2-2027H1 兑现。**收入构成**：综合价值包，含 DPU 卸载、低价服务器等，刨除直接 GPU 配套。**测算**：Vera 256 机架系统单价 ~$4M，200 亿美元 ≈ **4,000 台 Vera 256** ≈ 100 万颗 Vera CPU（每台 256 颗）。**单颗 Vera CPU 价格**：$8,500-$12,000（合理估值 ~$10,000）；不能用 Grace 类比（Grace 仅数据搬运辅助角色）。**未来 2 年市场**：GPU/ASIC 保守 1,000 万颗起，CPU:GPU = 4:1 → 需 4,000-5,000 万颗 CPU → 100 万颗仅起点。**英伟达数据意义**：CPU 进入 AIGC 驱动新历史周期，全行业（ARM/x86/CSP 自研 Graviton+Cobalt/Intel/AMD）受益。**边缘端拓展**：GB100 桌面 AI 等小型服务器配 Vera CPU 增量可观。台积电产能与 GPU 同样受限，交付时间跨度超 1 年。

## 关键数据点

- **Vera CPU 收入预期**：$200 亿（2026H2-2027H1）
- **Vera 256 系统单价**：$4M
- **Vera CPU 单价**：$8,500-12,000
- **对应出货量**：~100 万颗 (4,000 台 × 256)
- **未来 2 年 CPU 需求**：4,000-5,000 万颗
- **GPU vs CPU 比例**：4:1
- **Vera 256 vs 32 路传统**：等效 3 万多台
