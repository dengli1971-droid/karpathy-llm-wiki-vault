## 核心摘要

申菱视角液冷一次侧大厂调研。**液冷系统层级**：水冷背板/CDU/冷板（小颗粒）+ 一次侧干冷器/冷水主机/冷却塔（机组级）；申菱专注机组集成（管路自做），冷板/UQD/板换/水泵外购。**1GW 数据中心价值量**：风冷 ¥18-20 亿 → 液冷干冷器方案 ¥35 亿（+75-94%）/ 冷却塔+水冷主机方案 ¥40 亿（+100%）。**两条一次侧路线分化**：①北美传统=闭式冷却塔+冷水主机（机柜密度高+气候热+30℃环境需中低温水）；②国内=干冷器（性价比 + 80-120kW 机柜密度 + 西北寒气候 freecooling）；**北美现也推广干冷器**。**价值量差异**：1GW 干冷器 ¥7-9 亿 / 冷却塔 ¥13 亿 / 冷却塔+水冷主机 ¥30 亿；冷水主机系统复杂稳定性高 → 技术壁垒 + 溢价高。**地区分布**：东南亚=冷却塔+水冷主机（必须，气候热）/ 国内干冷器=西北 / 北美倾向冷水主机。**总结**：干冷器 ¥35 亿 < 冷却塔方案 ¥40 亿（贵 30-40%）。

## 关键数据点
- 1GW IDC：风冷 ¥18-20 亿 → 液冷干冷器 ¥35 亿 / 冷却塔水冷 ¥40 亿
- 一次侧组件价值量：干冷器 ¥7-9 亿 / 冷却塔 ¥13 亿 / 冷水主机+塔 ¥30 亿
- 国内 IDC 西北气候 + 干冷器 freecooling
- 北美机柜密度高 → 必用中低温水 → 冷水主机+塔
- 东南亚必用冷却塔+水冷主机
- 冷水主机方案 vs 干冷器贵 30-40%
