## 核心摘要

谷歌内部视角搜索+ Gemini+Agent 战略。**传统搜索超预期增长**（2026Q1 强、Q2 高于正常年）三驱动 = 追问增多（在传统搜索框）+ AI Overview UX 提升 + 可搜索界面/意图扩展（编程类查询前 3-4，免费 Flash 3.5 表现亮）。**AI 战略核心**=构建 Agent 基础设施嵌入垂直场景。**Agent 形态二**：①直接交互（如打车）②异步执行（监控价格/票务，使用量更大）。**优先级**：①**购物 Agent 最高**——商家上传 feed+优惠券+支付 → 直接闭环；推动力 = 高价值商业流量被 Amazon/Meta 分流；Commerce Center 积累 catalog；②本地生活——Maps 商家信息全；曾试水管工担保支付一两年前停（验资难）；③**Uber A2A 集成已在德国试两个月（限 Samsung+Pixel 高端机）**。**Chrome 内置 Agent**：跨标签连续操作数百步 + 99% 准确率（业内里程碑）。**Gemini 编程**：vs Opus 4.7 半代差距，免费 Flash 3.5 最强。**AI Overview**：当前稳态推国际化。**搜索拒激进**：保留链接/来源 + 广告订阅双轨延续。**Spark 智能体**：2026 夏上线，日历自动管理，分步渐进 + 防黑客边界。**对外开放 TPU**：自用+对外（含竞品）借需求反向迭代+规模经济。

## 关键数据点
- Chrome Agent：跨标签连续数百步 / 99% 准确率
- AI Mode 编程类查询前 3-4 位
- Gemini Flash 3.5 = 免费档最强
- Uber A2A 德国试用 2+ 月（限 Samsung+Pixel 高端机）
- Q1 增速高于 Q2（H1 整体显著高于正常年）
