## 核心摘要

**核心命题**：需求真实 ≠ 利润真实。**消费者剩余 ≠ 股东回报**。芒格框架问的不是"哪些行业有需求"，而是"行业内部的竞争会不会把价值创造转化为股东回报"。**主战场**：企业抢增量需求时最有效的武器是降价还是差异化——降价的行业容易变坏（短期理性叠加=行业非理性），差异化的行业更可能留下利润（品牌/口味/渠道/信任/审美/身份/习惯/生态绑定/网络效应/迁移成本）。**五要素叠加判别破坏性**：①消费者是否在乎是谁提供 ②固定成本高/边际成本低（闲置驱动"高于边际就卖"冲动）③扩产容易（资本快速进入）④份额是否改善单位经济（补贴买来的=有毒）⑤竞争者是否理性（融资/卡位/KPI/地方就业目标 vs 利润）。**最容易让投资者受伤的不是"明显没需求"，而是"需求很好但竞争很坏"的行业**——叙事漂亮（空间巨大/政策/渗透/技术进步），所有人都看见、所有人都融资扩产、所有人都信自己是最后赢家 → 行业进入"需求增长 + 收入增长 + 出货增长 + 利润越来越薄 + 现金流越来越紧 + 资产负债表越来越重"。**历史案例**：光伏 / 面板 / 锂电材料 / 通用制造 / 低端电商 / 本地生活补贴战。**最终问题**：行业的进步会被谁拿走（消费者降价 / 下游议价 / 新进入扩产 / 平台补贴）。**好生意 = 让利后仍能通过品牌/效率/网络/客户关系/定价权把利润拿回来**。

## 关键金句
- "需求真实不等于利润真实"
- "竞争的结果是把价值创造转化为消费者剩余，还是股东回报"
- "最危险的行业 = 需求很好但竞争很坏（叙事漂亮）"
- "行业长大后钱会留在谁手里"
