## 核心摘要

**MSAP 工艺正全面替代 HDI 在高端**：线宽 25-35μm（HDI 50μm）；800G 光模块 100% 用 MSAP，谷歌 V7 / iPhone 18 / 小米 YU7 / 问界 / Rubin Ultra/Feynman / 新 Windows 电脑 / 244GHz+ 信号 全部用。**渗透率演进**：当前 HDI 65% / SLP 10% / MSAP 25% → 2027 HDI 50% / SLP 5% / **MSAP 45%** → 2028 HDI 15% / SLP 15% / **MSAP 70%**（高端基本取代 HDI）。**梯队**：第一 鹏鼎/深南/沪电 → 第二 胜宏/方正 / 兴森（800G 能但 1.6T 不能）。**追赶壁垒**：层压 + 蚀刻工艺；胜宏为追赶包下欧姆龙 X 光对位机产能（一线 2 台 vs 追赶者 400 台）；新进入者 2 年常规 / 半年挖人；MSAP 资本开支 ¥1.5-2 亿/万 m²。**400G 光模块价值量**：2 年前 ¥18,000/m² → 现 ¥4,600/m²（国内一线大厂停做转东南亚）。**800G 光模块**：HDI 良率 <80% vs MSAP 88-92%；价值量 ¥12,000-14,000/m²。**1.6T 光模块**：2026/1 ¥32,000 → 当前 ¥41,000-43,000/m²（基板涨 30% + 供不应求）；线宽 20μm 必须 MSAP；3.2T 线宽 10μm。**关键设备**：① **电镀**（亚洲 ¥2,800 万/线，15μm 以下 / 东威 ¥2,000-2,200 万，15-35μm；亚洲年产能 120-150 + 东威 180-200，2025/10 起预订满，下单至交付 **1.5 年**）② **LDI 曝光机**（ORC 以色列 + 日本 + 国产芯碁微装，¥1,100-1,300 万，交付 4-5 月，国产不输海外）③ **激光钻孔机**（三菱主导 + 大族对胜宏战略合作便宜但精度差，三菱 50μm 极限 / 大族 100-110μm，三菱单机 ¥700-800 万，交付 8 月）。**MSAP 钻孔要求**：HDI 孔径 0.2-0.25 / 孔间距 100μm；MSAP 孔径 0.1-0.15 / 孔间距 50μm 甚至 40μm；10μm 偏差就 CAF 离子迁移漏电。**3.2T 载板**：10μm 制程 + 孔径 0.1 = 钻针纯极限。**钻针寿命**：HDI 一支钻 2,500 孔 vs MSAP 仅 400-500 孔（**5×** 损耗）；2027 仅 0.1/0.15/0.25mm 三种钻针 **月需求 18-20 亿支**。**钻针价格**：PCD 金刚石 ¥2,000/支 / 0.15mm 白针 ¥40。**钻针格局**：鼎泰高科 0.1mm 垄断 + 0.15mm 领先 + 金洲（0.25mm 主，0.2mm 尚可，0.15mm 不行）+ 尖点（=金洲）+ 厦门夏芝（>0.35mm 才行）。**高端铜箔双寡头**：三井 vs 德福（中国），载体铜箔 ¥21 万/吨 / HVLP4 ¥19 万 / 普通 ¥9 万；德福已过英伟达+谷歌认证；进入壁垒微裂纹工艺，铜箔是英伟达三阶段认证第一阶电信测试；扩产无设备瓶颈但工艺管控难。**电子布扩产瓶颈是日本丰田 90% 织布机不放手**（vs 铜箔靠工艺）。

## 关键数据点

- **MSAP 渗透率**：2026 25% → 2027 45% → 2028 70%（高端）
- **PCB 工艺线宽**：HDI 50μm / MSAP 25-35μm / 1.6T 20μm / 3.2T 10μm
- **MSAP 良率**：800G 88-92% vs HDI <80%
- **PCB 价值量**：400G ¥18,000→¥4,600 / 800G ¥12-14K / 1.6T ¥32K→¥41-43K /m²
- **设备价格**：亚洲电镀 ¥2,800 万 / 东威 ¥2-2.2K 万 / LDI ¥1,100-1,300 万 / 三菱激光 ¥700-800 万
- **设备交付**：电镀 1.5 年 / LDI 4-5 月 / 激光 8 月
- **MSAP 钻孔**：孔径 0.1-0.15 / 孔间距 40-50μm
- **钻针寿命**：HDI 2,500 vs MSAP 400-500 孔（5× 损耗）
- **2027 钻针月需求**：18-20 亿支（0.1/0.15/0.25mm）
- **钻针价格**：PCD ¥2,000 / 白针 ¥40
- **高端铜箔价格**：载体 ¥21 万/吨 / HVLP4 ¥19 万 / 普通 ¥9 万
- **MSAP 资本开支**：¥1.5-2 亿/万 m²
- **钻孔机转速**：大族 25 万转 / 其他 28 万转；孔瞬温 280-320°C
