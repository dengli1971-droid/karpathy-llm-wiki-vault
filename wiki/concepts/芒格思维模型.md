## 定义

**芒格思维模型**：以"行业内部竞争结构 → 价值最终归属"为核心的投资判别框架。区别于流行的"赛道空间/增速/份额"三件套，芒格框架强调：**需求真实 ≠ 利润真实；规模扩大 ≠ 价值扩大；份额提升 ≠ 护城河**。判断一门生意时，问的不是"行业能否长大"而是"行业长大后钱会留在谁手里"。

## 关键信息

### 框架核心问题（"重读芒格 一/三"两层）

**Q1：行业增长 → 利润归属**
- 需求真实但 ROIC 烂 = 价值被消费者通过降价/下游议价/新进入扩产/平台补贴消耗掉
- 例：航空 / 光伏 / 锂电材料 / 通用制造 / 低端电商 / 本地生活

**Q2：份额是护城河还是毒药**
- 好份额 = 单位经济变好（单成本↓ + 黏性↑ + 议价↑ + 变现多样化 + LTV↑）
- 毒份额 = 用户忠诚补贴 + 规模反带复杂度 + 不可涨价/抽佣 + 可被同样补贴复制 + 内部 KPI 被规模绑架

### 破坏性 vs 保护性竞争（决定行业利润分配）

| 决定变量 | 破坏性方向（利润流出股东） | 保护性方向（利润留股东） |
|---|---|---|
| 抢客户武器 | 降价 | 差异化 |
| 消费者偏好 | 只比价格/功能 | 在乎品牌/信任/习惯/身份 |
| 固定成本 | 高（产能闲置压力大） | 低 |
| 扩产门槛 | 低（资本可快速复制） | 高 |
| 份额来源 | 补贴 | 网络效应/规模效应 |
| 竞争者目标 | 规模/融资/卡位/KPI | 利润/ROIC |

### 五个尖锐问题（投资本质）

1. 这个行业的进步最后会被谁拿走？
2. 钱会留在谁手里？
3. 资本市场是否还愿意给亏损玩家继续输血？
4. 规模变大后，单位经济模型有没有变好？
5. 用户/商户/供应链的忠诚是给品牌还是给补贴？
