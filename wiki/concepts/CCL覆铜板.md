## 定义

**CCL（Copper Clad Laminate, 覆铜板）** 是 PCB 的核心基板，由树脂浸渍玻璃布 + 铜箔层压而成。AI 服务器主板的关键约束是高速信号完整性（Dk/Df 介电损耗），驱动 CCL 等级从 M4 → M6 → M7 → M8 → M8.5 → M9 持续升级。Rubin 平台标志着 M8.5/M9 成主流，对应 **Low Dk 二代玻璃布** + Q 布 + HVLP 铜箔 + 特种 PPO 树脂的协同升维。

## 关键信息

### 等级与价格阶梯

| 等级 | 价格（元/平米） | 应用平台 |
|---|---|---|
| FR4 普通 | 160-170 | 消费电子 |
| 普通高速 | 200+ | 通用通信 |
| M6 | ~250 | GB 前代 |
| M7-M8 | 数百-1,000 | GB 系列 |
| M8.5-M9 | 1,000+ | Rubin |
| ≥M9 | 待定 | Rubin Ultra |

### 关键原材料瓶颈

- **Low Dk 二代玻璃布**：国内国际复材独家通过认证，月供 50-60 万平米；价格 25 年底 80-90 → 100+ 元持续上涨
- **HVLP 铜箔**：日韩三井/古河/金居/福田为主，国产化能力不足
- **特种树脂 PPO**：稳定但成本传导通畅
- **二代玻璃布生产门槛**：1,500-1,600℃ 池窑 + 丰田织布机 + 高目数漏板，新厂建产线需 1-2 年

### 产能折算

CCL : 玻璃布 ≈ 1 : 3-4 平米；72 卡机柜 ~150 平米 CCL → 国际复材月供 50-60 万平米玻璃布 → 1.5 万机柜/年支撑

### 成本传导

- 卖方市场 → CCL 涨价顺利向 PCB → 终端客户传导
- 材料成本在整机价值占比低，下游接受度高
- 国产替代窗口期：2026-2027 Rubin 量产 + 国产二代玻璃布认证爬坡

## Rubin Ultra 正交背板与 PTFE 路线（2026-06-07 更新）

**Rubin Ultra（NVL576 die / Kyber 机架 / H2 2027）正交背板单块 78 层 × 1.2m × 0.2m × 1cm 厚（量产无人区）**；单机柜 PCB 子系统价值 11.7 万美元（vs GB300 3.5 万 / Hopper 1 万，+10x）。**224Gbps PAM4 = 56GHz 奈奎斯特 + 趋肤深度 0.28μm** 倒逼三大材料升级：(1) HVLP4/HVLP5 镜面铜箔；(2) **PTFE 树脂登场**（Dk 2.1-2.6 / Df 0.0002-0.001 vs M9 Dk 3.0-3.5 / Df 0.002-0.004）；(3) Q 布配套但加工困难。**三方案博弈**：M9+Q 布（极致性能但 Q 布钻嘴 100-200 vs FR-4 1500-2000）/ **M8+PTFE 性能差 3-4% 但成本 <1/3、加工最难良率最低** / M10 PTFE 实验室。**PTFE 三大工艺地狱**：冷流性翘曲（320℃+）/ 孔壁润湿金属化（化学惰性 + 钠-萘络合物或等离子体活化）/ 钻孔拉丝胶渣 + HVLP4 镜面铜箔咬合力下降爆板。**Q 布日东纺接近垄断**（高端 T-glass）+ 2025/8 扩 3 倍但 2027-28 释放，2026 NVDA 需求 400-500 万米/月远超有效产能；**M10 PTFE 树脂 = 大金/罗杰斯把控**（金属离子 ppb 级），圣泉 M9 碳氢百吨级 / 东材+迪赛 BCB 千吨级试产。**关键验证窗口 = 2026 Q3-Q4 NVDA code 批准 + 2026 Q4-2027 Q2 PTFE 良率 50-60% → 75%+**。
