### 关键产业洞察

#### 先进封装成为 AI 芯片的"硬约束"

**产能缺口是最紧张的**：
```
当前瓶颈排序：
1. CoWoS 封装产能（最紧张）
   ├─ 台积电月产能：7.5-8万片 (2025年底) → 12-13万片 (2026年底)
   ├─ 外包量：24-27万片 (2026)
   └─ 供不应求至 2027-2028 年
2. HBM 晶圆产能（次紧张）
3. GPU 设计能力（充足）
4. 光互连（持续紧张但相对宽松）
```

**含义**：
- 超大规模芯片的制造与互连，最后的卡脖子竟然是封装
- 这给 OSAT 行业带来了"扩产+提价"的黄金期
- CoWoS 从"后道工序"变成"百亿美元独立利润池"

#### 台积电 CoWoS 外包的分配逻辑

**Amkor vs SPIL（日月光北美）的竞争格局**：
```
2026 年外包产能分配（24-27万片 CoWoS）：
├─ Amkor：18-19万片 (68-70%)
│  └─ 特点：承接新兴 AI 客户（Google TPU、英伟达等）
│  └─ 地理：美国中部、菲律宾成本优势
│  └─ 增速：最激进的产能扩张
│
└─ SPIL（日月光北美）：6-8万片 (22-30%)
   └─ 特点：服务传统 CSP 和北美客户
   └─ 地理：美国 West Coast，靠近客户
   └─ 增速：相对保守，重品质而非量
```

**为什么 Amkor 能拿到更多份额**：
1. **成本优势**：菲律宾产线成本低于 SPIL 美国产线
2. **扩产速度**：Amkor 投资更激进，产能爬坡更快
3. **客户导向**：Amkor 对新兴 AI 客户（Google、英伟达）的优先级更高
4. **地缘政治**：美国本土化政策对 Amkor 的激励更强（美国企业 + 菲律宾低成本组合）

#### 全球 OSAT 的分化：多维竞争 vs 单客户专业

**竞争维度**：
```
Amkor + 日月光：多维全球竞争
├─ 客户多元化（全球 Top 10 芯片商都是客户）
├─ 地域多元化（全球产线布局）
├─ 技术多元化（CoWoS、HBM、硅光等全面覆盖）
└─ 利润最大化（通过规模、技术、地缘价值套利）

长电科技：全球客户 + 中国优势
├─ 客户多元化（通过 STATS 全球品牌）
├─ HBM + 硅光双引擎（差异化竞争）
├─ 成本优势（新加坡 + 中国产能组合）
└─ 前景：争取"技术型合作伙伴"而非"代工承接方"定位

通富微电：单客户专业化 + 新产线扩张
├─ AMD 单一客户（70%营收）
├─ MI 系列 AI 加速器专业化（与 AMD 深度合作）
├─ 汽车电子新增长点（分散风险的尝试）
└─ 风险：客户集中度在客户景气周期中的放大效应
```

#### NPO 市场的真实规模

**市场规模计算**：
```
Vera Rubin Ultra 机柜：单台 648 个 3.2T NPO 光引擎

出货规模（保守估计）：
├─ 2027 年：1 万台机柜 = 648 万只光引擎
├─ 2028 年：5 万台机柜 = 3,240 万只光引擎
├─ 2029 年+：10+ 万台机柜 = 6,480+ 万只光引擎
└─ 累计 3 年：≈ 1 亿只，约 $200-300 亿市场

对比光模块：
├─ 光模块（1.6T）：年 2,000-3,000 万只，市场价值 $100-150 亿
└─ NPO：年 2,000+ 万只（2028 年），市场价值 $200-300 亿（高端）

结论：NPO 是未来 3-5 年最大的光互连市场机遇
```

#### Feynman 架构的双轨制战略

**全光互联（CPO）的极端成本**：
- 光引擎集成在芯片，任何故障 = 整个 GPU 报废（$10,000+）
- CoWoS 集成复杂度极高，成本 $3,000-5,000/芯片
- 结论：暂不可行，只能用于极少数超高端场景

**铜光混合（NPO）的现实选择**：
- 机柜内铜线，跨机柜光纤（NPO）
- 光引擎故障可单独维护，不影响 GPU
- 成本 $2,000/光引擎（相对可控）
- 结论：**性价比最优，Vera Rubin Ultra 的标准**

**未来演进**：
- 短期（2026-2028）：NPO 为主流
- 中期（2029-2030）：CPO 开始试用（供应商品质提升）
- 长期（2031+）：全光互联成为主流（成本下降后）

- **不是技术差异**（旭创技术更强）

### 关键产业洞察

#### 供应链紧张长期存在
- **CW 产能瓶颈至 2028 年**：住友满产超 100% 无法增产；MOCVD 交期长（12-18 个月）
- **能做多少就能卖多少**：需求"无限大"，供应端完全受制于产能
- **国产替代 3-5 年周期**：源杰虽有竞争力但仍需时间认证与大客户导入

#### 大功率 CW 是价值所在
- **小功率 CW**：3-6 美元/颗，竞争激烈，利润薄
- **大功率 CW**：20-30+ 美元/颗，Lumentum 已放弃小功率只做大功率（利润 10 倍）
- **150mW 一分八**：关键创新，将 4 颗 70mW → 1 颗 150mW，单模块成本降 3-5 美元

#### 硅光成为主流
- **EML 竞争力下滑**：成本居高不下，难以降价；被硅光方案全面冲击
- **800G 硅光占比**：国内新招标基本都选硅光；EML 仅保留特定应用
- **成本曲线**：硅光的长期降价空间 > EML，通过功率升级与芯片数量优化

#### 英伟达 CPO 战略确认
- **Lumentum 被指定**：英伟达 CPO 光引擎明确指定采用 Lumentum CW
- **源杰有机会**：性价比优势，有潜力进入英伟达供应体系（备选或长期替代）
- **市场规模**：CPO 大功率 CW 市场 40+ 亿美金（到 2029），英伟达 CPO 占 50%+
