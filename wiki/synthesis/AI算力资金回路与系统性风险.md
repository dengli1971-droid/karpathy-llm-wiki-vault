# AI 算力资金回路与系统性风险

> **一句话内核**：AI 算力链表面是 Capex 周期，本质是一条 **NVDA → 4 CSP → Neocloud → 头部模型公司 → 又回到 NVDA** 的自我强化资金回路；当 NVDA 穿透后真实集中度突破 80%、CoreWeave 用 5-6 年期债买 2-3 年寿命 GPU、四大材料产能将在 2027H2 集中释放时，回路从"自反馈飞轮"变成"自反馈拆解器"的概率被卖方系统性低估。

---

## 一、本文要回答的六个问题（每一个都可以被反驳）

1. NVDA 70% 客户集中度红线已破到什么程度，"穿透"之后的真实数字是多少？
2. 把 CoreWeave 这类 Neocloud 当成 NVDA 的"独立客户"是否在统计上构成系统性误导？
3. 用 5-6 年期债买 2-3 年技术寿命的 GPU，飞轮何时变拆解器？触发点的财务量化指标是什么？
4. 2027H2 的所谓"拐点"是单一变量还是多变量共振？卖方为什么对此盲目？
5. 同时唱多 SST + HVDC + 液冷在逻辑上为什么互相矛盾？这反映卖方对 Capex 总盘子的什么误读？
6. 上面五点同时成立时，市场如何定价？6/01-6/07 的纳指 -5% 是不是回路松动的第一根 K 线？

## 二、资金回路的拓扑结构

### 2.1 三层嵌套图

```
                  ┌──────────────────────────────┐
                  │     NVDA (FY27Q1 数据中心 $752 亿)
                  │     单季 +85% / GAAP 净利 +200%+
                  └────────────┬─────────────────┘
                               │ 应收 / 营收口径
            ┌──────────────────┼──────────────────┐
            │                  │                  │
   ┌────────▼──────┐  ┌────────▼──────┐  ┌───────▼────────┐
   │ 第一层：4 CSP │  │ 第二层：Neo-  │  │  第三层：模型公司
   │ MSFT/GOOG/    │  │ cloud 代持     │  │  OAI / Anthropic
   │ META/AMZN     │  │ CRWV/Oracle/   │  │  → 通过 CW/Oracle
   │ 直购 75-78%   │  │ Lambda +10-15% │  │  又自购 GPU
   └───────┬───────┘  └────────┬───────┘  └────────┬───────┘
           │                   │                   │
           │ Capex 资金 ←──────┴───── 投资 OAI/Anth─┘
           │                                       │
           └──────────── Token 经济回流 ───────────┘
                  (字节 26 年 1,200 亿 Capex)
                  (谷歌 26 年 1,800-1,900 亿)
```

[[摘要-nvda-customer-concentration-70-2026-06-02]] 给出的关键穿透：
- **表面**：前 3 直接客户按收入 = 54%，按应收账款 = 64%。FY26 Q3 前 4 客户应收 = 65%。
- **穿透后**：CoreWeave 大客户实际是 OpenAI 和 MSFT；Stargate 7GW / $5,000 亿规模 → **OAI 通过"直购 + Oracle 转手 + CW 转手"三路径**对 NVDA 同时构成集中度风险。
- **真实终端集中度 ~80%**。

### 2.2 集中度的两个层级被卖方刻意混淆

| 口径 | 数字 | 卖方解读 | 真实含义 |
|---|---|---|---|
| 法律实体客户数 | 数百家 | "客户分散" | 没有意义 |
| 直接收入前 3 | 54% | "已经偏高但可接受" | 已破 50% 红线 |
| 直接收入前 4 应收 | 65% | "应收账款季节波动" | 集中度高于营收口径 |
| **穿透后终端用户** | **~80%** | 卖方拒绝穿透 | **70% 红线已实质击穿** |

70% 是历史经验上"任一单点冲击都能直接传导到营收"的容错阈值。**穿透后 80% 意味着 NVDA 已不存在抗冲击缓冲**。2022 年挖矿崩盘单季营收两位数下滑是无杠杆案例；今天的资金回路有杠杆，去化速度更快。

### 2.3 资金从哪里来：四个源头被高估了三个

| 源头 | 2024-2025 来源 | 2026 真实情况 |
|---|---|---|
| ① CSP 经营现金流 | 谷歌/MSFT/META 现金牛业务 | 谷歌 2026/6 一次性融资 847.5 亿（含 100 亿伯克希尔私募）= **现金牛不够用了**，开始动用股权 |
| ② 头部模型公司 | 风险资本 + CSP 投资 | OAI/Anthropic 烧钱速度上升，依赖 CSP 反向输血 |
| ③ Neocloud 信贷 | DDTL 1.0/2.0/3.0 银团贷款 | CW DDTL 4.0 $85 亿首次获 A3/A(low) **投资级评级** —— 但这本身是回路自我背书：用客户长约抵押向银行融资买 NVDA GPU 租给客户 |
| ④ 主权 AI | Stargate / 沙特 / UAE | **政策性一次性投入**，存续性存疑（[[摘要-ai-infra-2027h2-supply-demand-reversal-2026-06-06]] 直接指出） |

> **关键不对称**：①是真实外部资金，②③是内部资金转手（钱在回路内循环但被多次计数为"GDP"），④是政策驱动一次性。**真正能持续为回路注入新血的只有 ①**，但谷歌 26/6 融资恰恰说明 ① 已经吃紧。

参见 [[摘要-internet-cloud-pricing-2026-06-01]]：字节 2026 Capex ¥1,200 亿 / 采购口径 ¥4,500 亿，**50%+ 是内部成本费用化** —— 在中国端这个回路嵌套甚至更深。

## 三、CoreWeave 久期错配：回路最脆弱的节点

[[摘要-coreweave-duration-mismatch-2026-06-03]] 的金融本质拆解非常清晰：

### 3.1 三重错配

| 错配维度 | 资产侧 | 负债侧 | 含义 |
|---|---|---|---|
| **账面 vs 实际寿命** | 训练 GPU 经济寿命 2-3 年 | 账面折旧 6 年（2023/1 从 4→6 调整） | 净利润被会计延迟揭示 |
| **资产 vs 负债期限** | GPU 实质报废 2028 | DDTL 集中到期 2028-2032 | 还款日 = 资产无法变现日 |
| **GAAP vs 现金** | FY25 利息 24% / 折旧 48% | GAAP 净亏 -$11.7 亿 vs Adj EBITDA $30.9 亿 60% 利润率 | 会计利润是制度产物 |

### 3.2 CW 飞轮 = 1999 Global Crossing 同构 + 久期更短

```
低融资成本（DDTL 4.0 SOFR+2.25%）
   ↓
破坏性低租金（H100 现货 $8-10 → $2.5-3 / 跌 60%）
   ↓
Take-or-Pay 长约（Meta $352 亿至 2032 + Rubin / OAI 5 年 $119 亿）
   ↓
SPV 抵押融资
   ↓
更多 GPU 采购
   ↓
更多 NVDA 收入 → 拉高 NVDA 估值
   ↓
回到第一步
```

**vs 1999 GC 的关键差别**：CW 资产是 3 年技术淘汰 GPU，GC 资产是 20 年寿命光纤 —— **资本结构同构，资产生命周期更短，CW 的危险性远高于 GC**。

CW 唯一比 GC 强的地方是 Take-or-Pay 合约的法律约束力（IRU 是弱约束），但这个优势依赖一个前提：**客户履约能力本身不依赖于回路是否还在运转**。Meta $210 亿 / OAI $119 亿都是这样的 take-or-pay，但 OAI 是否能持续兑现，本身就由"是否有人继续给 OAI 融资"决定 —— 这就是回路的递归。

### 3.3 CW 模式的五项护城河有明确失效边界

| 护城河 | 当前状态 | 失效条件 |
|---|---|---|
| Backlog $668 亿 | FY25 数字 | 任意 1 个超大客户合约不续约 |
| 违约成本设计强 | take-or-pay 合约 | 客户自己破产或主动违约（OAI 现金流压力） |
| 6 年折旧获 GOOG/MSFT/Oracle 背书 | 行业默认 | 训练场景实际 2-3 年技术淘汰被市场重新认知 |
| A3 投资级评级 | 2026 取得 | 评级方法依赖大客户履约 |
| 客户多元化（MSFT 62% → 两三家） | 已演变 | "两三家"仍是高集中度，且这些客户互相重叠 |

**最关键的脆弱点**：[[摘要-coreweave-duration-mismatch-2026-06-03]] 明确指出 —— **DSCR 1.4x 硬约束自 2027/4 起生效**。这是合约里黑字白纸的强制性现金流测试，CW 必须证明经营现金流能覆盖债务本息 1.4 倍。结合 GPU 现货租金跌 60%、2028 起 NVDA 进入 Rubin/Feynman 双代迭代，DSCR 测试很可能成为 2027 年的"白纸黑字触发器"。

## 四、2027H2 拐点：四个变量共振，不是单一变量

[[摘要-ai-infra-2027h2-supply-demand-reversal-2026-06-06]] 是这篇综合分析的支柱：**2026 北美 4 云 CapEx >7,250 亿（+77%），但 2027 后骤降至 10-17%**。这不是一个预测，而是四条不同链路推导出的共振。

### 4.1 四大产能瓶颈在 2027H2 同时缓解

| 材料/工艺 | 当前紧张 | 扩产 Lead Time | 释放时点 |
|---|---|---|---|
| **ABF 载板** | 味之素 26/5 提价 30%，缺口至 2027 底 | 3-4 年（2024-25 启动） | 2027-28 集中投产 |
| **HBM** | SK 50% + 美光目标 25% / Q1 DRAM +90-95% | 3 巨头 26H2-27H1 显著跃升 | 2027 阶段性过剩 |
| **CCL 二代玻璃布** | 台光 M9 NVDA 独家、26-28 提价 8-15% | 2-3 年 | 2027-28 议价权回头部需求方 |
| **T-glass** | 日东纺垄断 | 27-28 释放 | 2027-28 垄断松动 |

参见 [[摘要-hbm-tsv-equipment-leadtime-2026-06-07]]：美光台湾 HBM 月产能 2-3 万 → 2029 年 10-11 万（4 倍跃升）；这条扩产曲线一旦在 2027-28 兑现，HBM 立即从瓶颈变结构性过剩，类似 2022-23 DRAM 周期但来得更猛烈（因 3 大厂同时挤压力大）。

### 4.2 财务折旧约束 = 反向飞轮

| 变量 | 数字 | 含义 |
|---|---|---|
| 4 云 2024 资本存量 | 9,000 亿美元 | 直线法 5-7 年摊销 |
| 2027 折旧 | 2,200-2,500 亿/年 | **几何级数侵蚀净利润** |
| 云收入增速平台期 | ~30% | 增速 vs 折旧增速错位 |
| 推论 | ROIC 破坏 | 股东压力被动踩刹车 |

**关键内在矛盾**：[[摘要-capex-depreciation-tsmc-sk-samsung-2026-06-02]] 论证台积电 30-50% Capex/营收强度内生化为竞争方式，**这套逻辑对台积电成立，对云厂商不一定成立** —— 台积电的扩产对象是有 25-30 年寿命的厂房；云厂商的扩产对象是 5-7 年寿命的服务器 + 3 年寿命的 GPU。**折旧吞噬利润的速度对云厂商比对台积电快至少一倍**。这是 [[Beta_Alpha锚点框架]] 的应用：把"台积电高 Capex 通胀也能消化"的经验外推到云厂商是错的，因为底层物理（资产寿命）不同。

### 4.3 需求侧 ROI 临近交卷

- 2024 投入对应 2026-2027 交卷
- 推理替代训练主导 → Performance/Watt 取代堆矿石 → **算力单价压力上升**
- 主权 AI 政策一次性 → 一旦预算用完无续期机制

参见 [[摘要-ai-server-market-size-2026-06-03]]：2026 全球 AI 服务器 $4,000 亿 / 400 万台，2027 $5,000 亿 / 500 万台 —— 增速 25%（从历史 +64%/+77% 急剧降速）。**这个降速本身就是市场对 ROI 临近交卷的提前定价**。

### 4.4 卖方为什么对此盲目：线性外推的认知陷阱

- 2000 年互联网光纤泡沫前 18 个月，券商也在线性外推光纤 Lead Time 3 年缓解、需求继续翻倍
- 2026 年的故事几乎一致：把 +77% Capex 直接外推到 +50%/+30%，假定供给永远跟不上
- **真相**：四个材料一旦同时释放，叠加折旧反向飞轮，定价权 12 个月内换手

## 五、卖方共识的方法论漏洞：SST + HVDC + 液冷不能同时唱多

[[摘要-ai-bear-case-sst-hvdc-cooling-2026-06-06]] 这篇是这次批次最尖锐的反共识深度。**三者在物理上是规模依赖的排他替代关系，不是并驾齐驱**。

### 5.1 三档 TCO 临界

| 园区规模 | 主场技术 | 关键经济 | SST 是否成立 |
|---|---|---|---|
| **100MW LDT** | LDT (低压直流) | ¥1/W vs SST ¥7-8/W；100MW 初始差 ¥6-7 亿 / SST 回本 30 年+ | **不成立**（财务破产） |
| **500MW HVDC 黄金区** | 800V HVDC + 巴拿马电源 | 停机损 = 每小时几十-几百万美元 / HVDC 降维护 90% / SST 摊薄不够 | **不成立**（被巴拿马挤压） |
| **1GW 北美 Hyperscaler** | SST 直出 800V + 浸没液冷 | 物理空间机会成本主导 / 1GW 算力百亿美元 / SST 释放 10% IT 空间 = 多部署数万 GPU = 年增数十亿收入 | **唯一成立场景** |

### 5.2 时间维度的非对称

| 技术 | 商业化 | 投资含义 |
|---|---|---|
| 800V HVDC | **2025 起量元年** | 5 年 CAGR 高两位数 |
| 液冷 100% CDU 冷板 | 2025-26 标配 | 与 HVDC 共生 |
| **SST** | **2030-31 规模化营收** | **作为近期催化剂是严重时间错位** |

[[摘要-ai-bear-case-sst-hvdc-cooling-2026-06-06]] 的核心论断：**卖方把 SST 当作 2026-27 推荐，违反了对负载端（GPU/ASIC PSU 架构代际切换）必须先行的物理约束**。SST 商业化前提是 GPU/ASIC 原生支持 800V 直流，这需要 PSU 架构代际切换 —— 等不到 2030 年。

### 5.3 "卖方同时唱多"暴露了什么

**这不是策略错误，这是认知错误**。卖方逻辑链条是：
1. AI Capex 大 → 总盘子大
2. 每个细分都吃到一点 → 都能涨
3. 不深究互相排他

**正确逻辑链条（[[芒格思维模型]] 的破坏性竞争视角应用）**：
1. AI Capex 是有限蛋糕（即使 7,250 亿也有边界）
2. SST / HVDC / LDT 在每个规模档位上是排他选择
3. 同时唱多 = 没看到产业链内部的零和关系
4. 这种"百花齐放"叙事 = 1999 年所有光通信公司都涨的同一种认知错误

把这个框架延伸到本批次另一组明显矛盾：
- 一边推 NVDA 集中度无忧（[[摘要-nvda-customer-concentration-70-2026-06-02]] 反共识第一）
- 一边推 CRWV 飞轮成立（[[摘要-coreweave-duration-mismatch-2026-06-03]] 反共识第二）
- 一边推所有产业链环节都涨

**三者中至少两者矛盾**：NVDA 终端集中度 80% = CW 在 NVDA 客户名单里占很大比例 = CW 飞轮成立等价于 NVDA 集中度风险加大。卖方需要 NVDA 和 CW 同时成立，但同时成立的代价是承认 80% 集中度 = 单点风险被自身放大。

## 六、回路裂缝的早期定价：6/01-6/07 是不是第一根 K 线？

### 6.1 [[摘要-cyclic-extreme-ai-rotation-2026-06-07]] 的诊断

**05/17 后 AI 链 FOMO 没结束而是旋转**：
- INTC 从 4/24-5/15 +63% → 5/18-6/03 仅 +3.6%（动能耗竭）
- ARM +97% / DELL +74% / MRVL +71% / HPE +67% / SMCI +53%（资金向 AI 基础设施"交付层"集中）

**关键诊断**：每一轮"搜索下一个没涨够的节点"= FOMO 末期特征；每轮利好强度必须升级（DELL 需 +88% 营收才驱动新涨）= **耗竭信号**。

**估值定价已经前置完美**：
- ARM 170+ 倍 PE = 定价"AGI CPU 量产 + x86 线性替代 + RISC-V 不兑现 + SoftBank 不稀释"四个条件全成立
- DELL 167B 全年指引（含 60B AI 服务器）= 每季度都要 Q1 +757% 的强度

### 6.2 6 月第一周纳指 -5% 的归因（[[摘要-global-index-decline-w1-june-2026-06-07]]）

六因素分主次：
1. **谷歌 847.5 亿股权融资** — 背景（资金供需想象松动）
2. **博通业绩未超预期到足够好** — 拥挤交易踩踏触发器
3. **Rubin NVL72 CPU 侧 SOCAMM/LPDDR5X 砍 50%**（55TB → 28TB） — 内存链估值杀
4. **世界杯 6/11 情绪转移** — 辅助
5. **5 月非农 +17.2 万 / 加息预期 12 月前 52% → 68.4%** — **全市场主因**
6. **四巫日提前** — 放大波动

**最重要的 take**：
> "半导体新闻解释跌谁，非农和利率解释为什么一起跌"

这个判断把回路问题分成两个时间尺度：
- **短期（已发生）**：宏观利率 + 拥挤交易筹码重构
- **中期（待定）**：6/10 CPI 破 4% + Warsh 6/16-17 首次 FOMC 是两个分叉口

### 6.3 关键 take：6/01-6/07 是筹码重构，不是趋势破位 —— 但暴露了回路的薄弱

[[摘要-cyclic-extreme-ai-rotation-2026-06-07]] 的最终判断：**"行业趋势与股价之间隔着估值——AI 真实需求乘以情绪放大系数，放大系数比行业脆弱"**。

这句话翻译到回路语言：
- 回路本身的物理约束（GPU 短缺/Capex 增速/Token 需求）还在
- 但回路定价已经透支到要求"完美执行" + "无任何摩擦"
- **任何摩擦都会触发情绪去杠杆，先于物理拐点**

→ **2026/06 第一周很可能不是 2027H2 物理拐点的"提前定价"，而是估值放大系数自己的回归** —— 而这一回归会迫使整个回路对 2027 拐点重新加权概率。

### 7.2 触发链顺序

**最先触发**：⑤⑥（CSP Capex 增速降 + 折旧侵蚀利润）→ 卖方共识开始动摇
**第二批触发**：①③⑦（NVDA 集中度恶化 + GPU 租金继续跌 + HBM ASP 转负）→ 拐点确认
**最后触发**：④⑨（CW DSCR 失守 + 失去投资级）→ 危机时刻

**时间窗口**：

- 2027H2：④ 可能（DSCR 硬约束 2027/4 生效后第一个测算周期）

### 7.3 配 [[芒格思维模型]]：哪些 AI 应用是"真好生意"，哪些是"补贴撑场"

| AI 业务/资产 | 单位经济 | 是好份额还是毒份额 | 投资判断 |
|---|---|---|---|
| **NVDA GPU 销售** | 单卡毛利极高 / 客户被迫接受 | **好份额**（议价权 + 不可替代） | 风险来自集中度而非毛利 |
| **HBM（SK 海力士主导）** | TSV 设备交期锁死 / 准基建 | **好份额**（见 [[摘要-hbm-tsv-equipment-leadtime-2026-06-07]]） | 但 2027 周期反转 |
| **CoWoS（台积电）** | 工艺+客户黏性双绑定 | **好份额** | 长期成立，短期被集中度连带 |
| **CoreWeave GPU 租赁** | 单位经济见不到（GAAP 净亏） | **毒份额**（"用更多资本维持更大亏损池"芒格三精确描述） | 高久期错配风险 |
| **MaaS Token 销售（字节）** | 视频 12% / 文本 5-6% / 综合微利-负 | **毒份额**（[[摘要-internet-cloud-pricing-2026-06-01]] 显示存量综合毛利负或微利） | 单位经济未跑通 |
| **CSP 通用算力租赁** | 5 年长约毛利 25%+ | **好份额** | 但依赖客户长期履约 |
| **国产 ASIC（M890 等）** | 真武单卡 6.3-6.5 万 / 客户两三百家 | **好份额**（议价权差但增量收入清晰） | 政策周期叠加 |

**芒格框架的最尖锐应用**：
- **NVDA 表面是好份额，实际接近毒份额** —— 因为 80% 客户集中度本身就是议价权被反噬的形式（CSP 们联手反向定价）
- **CRWV 是教科书级毒份额** —— GAAP 净亏 -$11.7 亿、靠"更多资本维持更大亏损池"、客户对它没忠诚（OAI 一旦自建产能立即流失）

## 八、反方论据与开放问题

### 8.1 多头能给出的最强 counter-argument

1. **"AGI 临近 → ROI 不需要传统财务计算"** —— 反驳：所有泡沫论的最后一站。即使 AGI 在 2028 兑现，回路里的 CW/Stargate 的债务期限和 DSCR 是合同性硬约束，不会因为 AGI 兑现就豁免。
2. **"NVDA 集中度高但客户都是超级稳健大公司"** —— 反驳：穿透到 OAI 时代，OAI 的"稳健性"本身就是回路递归。
3. **"2027 HBM 过剩之前还有 18 个月窗口"** —— 反驳：成立，但 18 个月的高估值是定价完美执行，任何摩擦都触发去杠杆（见第 6 节）。
4. **"美国政策上不可能让 NVDA / CRWV 倒"** —— 反驳：政策能托底大客户，但不能托底估值。2000 电信泡沫崩盘后 CSCO/INTC 依然存在，但股价跌 80%。
5. **"主权 AI 是真实需求"** —— 反驳：[[摘要-ai-infra-2027h2-supply-demand-reversal-2026-06-06]] 已警告政策性一次性投入存续性存疑。

### 8.2 数据缺失与口径分歧

- NVDA 穿透客户集中度无官方披露，需调研侧穿透 → 数字 ±5pct 区间
- OAI / Anthropic 现金消耗速度无公开数据 → 只能通过 CSP 投资和招聘信号反推
- CW 5-6 年期债的真实再融资能力（市场 vs SPV vs 银团）需要更细颗粒度
- 2027H2 拐点是供给侧释放还是需求侧降速主导，分歧大（本文倾向 **二者共振而非一者主导**）
- 主权 AI（沙特、UAE）的资金充裕度与政策延续性是关键变量但难量化

### 8.3 wiki 当前没解决的开放问题

1. **国产算力链是否独立于这条回路**？ → 部分独立（资金来自国内 CSP 补贴 + 政策），但终端 token 需求仍受全球 AI 应用 ROI 影响 → 部分相关。
2. **Token 经济崩塌的临界点在哪**？字节日 token 5T → 1,000T 增 200 倍但毛利 5-12%，[[摘要-internet-cloud-pricing-2026-06-01]] 显示存量综合毛利负或微利 —— 单位经济何时跑通？
3. **GPU 二手市场是否会形成**？这是 CW 久期错配的最大不确定项，但目前可观测数据稀少。
4. **国产替代节奏会被 2027 拐点加速还是减速**？加速假说：海外缺货红利消失 → 国产被迫提速；减速假说：2027 全球过剩 → 海外低价倾销国产链。本 wiki 倾向 **取决于政策与海外去库存的相对速度**。
5. **CPO/NPO 在 2027 拐点中是缓冲器还是加速器**？这是 NVDA 战略侧的关键变量，需要后续综合分析。

## 九、本篇综合分析的最大 Alpha 命题

> **如果穿透后 NVDA 集中度 80% 已成立、CW DSCR 1.4x 硬约束在 2027/4 启动、四大材料产能 2027H2 集中释放这三件事同时被市场认知，资金回路的去杠杆速度会比 1999 光纤泡沫快至少一倍 —— 因为 GPU 资产生命周期是光纤的 1/7，会计延迟揭示的速度是物理变现的关键变量。**
>
> **逆向 Alpha 命题**：当前估值定价了"四大材料永远不缓解 + 集中度永远不去化 + 折旧永远不侵蚀利润"三个完美条件同时成立 —— 这三个条件中只要 1 个崩塌，回路即从飞轮变拆解器。三个之中 **最可能率先崩塌的是材料侧（确定性最高）**，**最迟崩塌的是集中度（最依赖市场认知）**，**最致命的是折旧（一旦显现就反向几何级数）**。
