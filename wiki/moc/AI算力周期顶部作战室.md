## 一句话内核

> **AI 算力链 = NVDA → 4 CSP → Neocloud → 头部模型公司 → 又回到 NVDA 的自反馈资金回路。当 (1) 穿透集中度 80% (2) CoreWeave DSCR 1.4x 硬约束首测 (3) 四大材料 2027H2 产能集中释放 三件事同时被认知，飞轮 → 拆解器，且因 GPU 资产寿命是光纤 1/7，去杠杆速度比 1999 光纤泡沫快至少一倍。**

## 三、多空双方对位

### 🔴 空头骨架（[[AI算力资金回路与系统性风险]] 九节框架）

1. **资金回路四源头三高估**：① VC ② Hyperscaler Capex ③ Neocloud 负债 ④ 政府补贴 → 只有 ① 是真正外部新血，①已吃紧（[[摘要-coreweave-duration-mismatch-2026-06-03]] 谷歌 26/6 融资信号）
2. **NVDA 穿透集中度 ≠ 申报集中度**：申报 ~70% / 真实穿透后可能 > 80%
3. **CW 久期错配**：5-6 年期债 vs 2-3 年寿命 GPU vs 客户 1-2 年合同 → DSCR 1.4x 硬约束 2027/4 首测
4. **四大材料 2027H2 集中缓解**：HBM / CoWoS / MLCC / CW 光源
5. **卖方同时唱多 SST + HVDC + 液冷是排他不并列**——三方案规模化时间和物理冲突

### 🟢 多头反驳（**待完整收集**——目前 wiki 里相关 source 偏少，是开放风险）

- 反方 1：AI 需求弹性远超供给侧担忧 → 但 ROI 临近交卷（[[摘要-claude48-codex-pressure-2026-06-06]] / [[摘要-token-pricing-tier-2026-06-03]]）
- 反方 2：Hyperscaler Capex 永续上行 → 但折旧 + 客户 ARR 临近 → 见 [[摘要-ai-infra-2027h2-supply-demand-reversal-2026-06-06]] 反方论点章节
- 反方 3：Rubin 平台 BOM 价值跃迁是结构性（[[Rubin平台BOM全产业链拆解]] 多头 Alpha 论）—— 注意这是被动元件/PCB 链路的多头，跟「NVDA 估值多头」不是同一件事，**要分清主体**

> ⚠️ **方法论提醒**（参考 [[Beta_Alpha锚点框架]] + [[芒格思维模型]]）：多头论点常常把「结构性 Alpha（[[Rubin平台BOM全产业链拆解]] 里 PTFE / 电容 / 钽电容）」和「周期性 Beta（NVDA 估值倍数 / 集中度持续）」混合，注意分层评估。

## 六、当前认知缺口（开放问题）

> 这些是 wiki 暂时没解决的, PM 决策前**必须心里有数**:

1. **多头反驳数据严重不足** —— 当前 wiki source 偏空头视角, 多头要点散落且不成体系. 建议下次研究主动找卖方多头报告填充.
2. **NVDA 穿透集中度的真实数据** —— "70% 红线已破" 是分析师推论, 没有公开穿透数据. 决策前需进一步核实.
3. **CW DSCR 1.4x 硬约束的合约文本** —— 来自 [[摘要-coreweave-duration-mismatch-2026-06-03]] 的解读, 但具体债券契约文本尚未读到. 关键决策前应直接查 SEC filing.
4. **2028 备料拐点的反向验证** —— 多家纪要都提"翻 3 倍备料", 但口径不同 (有的说 28 年, 有的说 27H2). [[Rubin平台BOM全产业链拆解]] 已标"待补"
5. **Capex YoY 拐头的实际催化剂** —— 是 ROI 证伪、HBM 缺货、地缘还是利率？多变量纠缠没拆清
